Determinant

The determinant of a matrix assigns a scalar value (number) to the matrix. For a 2×22\times22×2 matrix A=[abcd]A=\begin{bmatrix}a&b\\c&d\end{bmatrix}A=[ac​bd​], we have the following formula: det⁡(A)=∣abcd∣=ad−bc.\det(A)=\begin{vmatrix}a&b\\c&d\end{vmatrix}=ad-bc.det(A)=∣∣​ac​bd​∣∣​=ad−bc.

Easy? Well, the general formula (for an n×nn\times nn×n matrix) is much harder to grasp (do it at your own risk).

This will be useful when we solve systems of linear equations.

Addition and Subtraction

To add two matrices of the same size, we just add the corresponding elements in the matrices (those having the same position in the matrices). Subtraction is defined similarly.

For example:

The sum and difference of two matrices with different sizes are not defined.

Scalar Multiplication

To multiply a scalar value (a number) to a matrix, we just multiply each element in the matrix by that number. This is called scalar multiplication (this name should be self-explanatory).

For example:
3×[9821]=[3⋅93⋅83⋅23⋅1]=[272463]3\times\begin{bmatrix}9&8\\2&1\end{bmatrix}=\begin{bmatrix}3\cdot9&3\cdot8\\3\cdot2&3\cdot1\end{bmatrix}=\begin{bmatrix}27&24\\6&3\end{bmatrix}3×[92​81​]=[3⋅93⋅2​3⋅83⋅1​]=[276​243​]

Matrix Multiplication

How about multiplying a matrix by another matrix? This is more complicated, although the validity of matrix multiplication is looser: if we want to multiply an a×ba\times ba×b matrix by a b×cb\times cb×c matrix, the only condition is that bbb will be the same throughout, and it will give an a×ca\times ca×c matrix.

The element at (x,y)(x,y)(x,y) of the a×ca\times ca×c matrix will be the dot product of the xxxth row of the first matrix and yyyth column of the second matrix. The dot product will be the sum of the products of matching members. For example: (1,2,3)⋅(4,5,6)=1⋅4+2⋅5+3⋅6=32.(1,2,3)\cdot(4,5,6)=1\cdot4+2\cdot5+3\cdot6=32.(1,2,3)⋅(4,5,6)=1⋅4+2⋅5+3⋅6=32. Thus, an example of matrix multiplication will look like this: [123789]×[410511612]=[3268122266].\begin{bmatrix}1&2&3\\7&8&9\end{bmatrix}\times\begin{bmatrix}4&10\\5&11\\6&12\end{bmatrix}=\begin{bmatrix}32&68\\122&266\end{bmatrix}.[17​28​39​]×⎣⎡​456​101112​⎦⎤​=[32122​68266​].

The computation process is this complicated because this is how we do things in real life! For example, say Sparky took a test with one-point, two-point, and three-point questions, and he got 666, 444, and 222 questions correct in each part respectively. Then, his score is just the dot product of (1,2,3)(1,2,3)(1,2,3) and (6,4,2)(6,4,2)(6,4,2)! Here’s the computation: (1,2,3)⋅(6,4,2)=1⋅6+2⋅4+3⋅2=20.(1,2,3)\cdot(6,4,2)=1\cdot6+2\cdot4+3\cdot2=20.(1,2,3)⋅(6,4,2)=1⋅6+2⋅4+3⋅2=20.

Properties (for ease of calculation)

Matrices are structures like numbers, meaning they have properties as well, and they also share the same names as the properties of real numbers! If they don’t, we must’ve not defined OnO_nOn​ and InI_nIn​ in the previous reviewer! Anyway, here they are. Assume all capital letters other than OOO or III are matrices, and all small letters are numbers.

Addition and Subtraction

  1. Zero Property: A+O=A−O=AA+O=A-O=AA+O=A−O=A

Scalar Multiplication

  1. Distributive Property: k(A1+A2+A3+⋯+An)=kA1+kA2+kA3+⋯+kAnk(A_1+A_2+A_3+\dots+A_n)=kA_1+kA_2+kA_3+\dots+kA_nk(A1​+A2​+A3​+⋯+An​)=kA1​+kA2​+kA3​+⋯+kAn​
  2. Additive Inverse: O−A=−1×A=−AO-A=-1\times A=-AO−A=−1×A=−A (this is how we generate negative matrices!)

Matrix Multiplication

  1. Matrix multiplication is NEITHER commutative nor associative, meaning the order you use when multiplying them matters!
  2. Distributive Property: M(A1+A2+A3+⋯+An)=MA1+MA2+MA3+⋯+MAnM(A_1+A_2+A_3+\dots+A_n)=MA_1+MA_2+MA_3+\dots+MA_nM(A1​+A2​+A3​+⋯+An​)=MA1​+MA2​+MA3​+⋯+MAn​
  3. Zero Product: A×O=OA\times O=OA×O=O
  4. There are certain combinations AAA and BBB of non-zero matrices such that A×B=OA\times B=OA×B=O.
