The Very Basics

A matrix is a collection of numbers arranged in a grid, enclosed by square brackets. They are denoted by capital letters. Here are a few examples of matrices.
A=[123456]B=[123]A=\begin{bmatrix} 1 & 2 & 3\\ 4 & 5 & 6 \end{bmatrix}\qquad B=\begin{bmatrix} 1 & 2 & 3\end{bmatrix}A=[14​25​36​]B=[1​2​3​]C=[1245]D=[123]C=\begin{bmatrix} 1 & 2 \\ 4 & 5 \end{bmatrix}\qquad D=\begin{bmatrix} 1 \\ 2 \\ 3 \end{bmatrix}C=[14​25​]D=⎣⎡​123​⎦⎤​

Rows in matrices go left to right while columns go top to bottom. These describe the size of a matrix. Therefore, we have the following about the four matrices above:

  1. AAA has 333 rows and 222 columns, and is thus called a 3×23\times23×2 matrix.
  2. BBB has 111 row and 333 columns, and is thus called a 1×31\times31×3 matrix.
  3. CCC has 222 rows and 222 columns, and is thus called a 2×22\times22×2 matrix.
  4. DDD has 333 rows and 111 column, and is thus called a 3×13\times13×1 matrix.

Vectors

A vector is a matrix with either only one row or column. We have the following for n∈Nn\in\Nn∈N:

  1. A row vector of order nnn is a 1×n1\times n1×n matrix. For example, BBB is a row vector of order 333.
  2. A column vector of order nnn is an n×1n\times 1n×1 matrix. For example, CCC is a column vector of order 333.
  3. A square matrix of order nnn is an n×nn\times nn×n matrix. For example, DDD is a square matrix of order 222.

Here are a few more definitions concerning rows and columns in any matrix:

  1. The nnnth row vector of a matrix is the vector containing the nnnth row of that matrix.
  2. The nnnth column vector of a matrix is the vector containing the nnnth column of that matrix.

Elements

Like sets, a number in a matrix is an element (or entry). We denote the location of an element in a matrix as (r,c)(r,c)(r,c) where rrr is the row the element is in and ccc is the column it is in.
